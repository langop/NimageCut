版本控制：
git rebase
#当前目录下初始化一个不存在的git项目代码库（本地仓库）
git init Project_Folder_xxx
#当前目录下初始化一个存在的git项目代码库（本地仓库）
git clone Project_Folder_xxx.git

#操作命令：
git checkout -b lxj = git branch lxj + git checkout lxj
git checkout -b lxj remote/xxx（）

git commit -am 'comment' = git add . + git commit -m 'comment'（新增文件不能处理）
git commit --amend -m 'comment'（修改最近）
git branch -m now_name new_name（重命名本地分支名）


追踪处理：
git branch --set-upstream local_name origin/remote_name（建立新分支，并追踪）
git branch --set-upstram-to origin/remote_name（追踪当前分支）
git push -u origin remote_name（每次push，使用-u，修改追踪）


查看命令：
git status
git branch（查看本地分支）
git branch -a（查看本地分支和远程分支）
git branch -r（查看远程分支）
git branch show origin（查看远程分支的状态）
git branch -vv（查看本地分支与远程分支的追踪关系）
